## Folder Structure
- `data/raw/`: Original dataset files
- `data/processed/`: Cleaned datasets
- `notebooks/`: Jupyter notebooks
- `src/`: Python source code
- `reports/`: Reports and observations
- `figures/`: Generated plots
- `models/`: Trained models

## Lab 1
Lab 1 focuses on development environment setup, dataset loading, data inspection, and initial exploratory data analysis.

## Lab 2
In this laboratory, we learned how to transform multiple raw relational tables into a
single clean analytical dataset suitable for future Machine Learning experiments.
This dataset is called an Analytical Base Table, also known as an ABT.

## Dataset
Place the Olist dataset CSV files inside `data/raw/` before running the notebooks. Dataset source: [Kaggle - Brazilian E-Commerce](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
